ModSync Setup

Automated modpack synchronization for Minecraft

Operating System

Minecraft Launcher

1. Enter Mods URL

Provide the URL to your modpack configuration file.

2. Download & Setup ModSync

1. Press Win + R
2. Execute the command below
3. When the window closes, the path is automatically copied to your clipboard
powershell -Command "$p = Join-Path $env:USERPROFILE 'ModSync.exe'; Invoke-WebRequest 'https://github.com/Ritze03/ModSync/releases/download/v0.1.0/ModSync.exe' -OutFile $p; $p | Set-Clipboard"
wget https://github.com/Ritze03/ModSync/releases/download/v0.1.0/ModSync -O ~/ModSync chmod +x ~/ModSync echo "Copy this: ~/ModSync"

After running the command, press Ctrl+V to paste the path into this field.

3. Configure Your Launcher

Follow these steps for your launcher to set up the pre-launch command.

1. Select your modpack in Modrinth App
2. Click the Settings icon in the top right
3. Select "Launch hooks"
4. Enable "Custom launch hooks"
5. Paste the command below into "Pre-launch"
1. Right-click the modpack/instance and select "Edit"
2. Select "Settings" on the left side
3. Go to the "Custom Commands" tab
4. Enable "Override Global Settings"
5. Paste the command below into "Pre-launch"
[ModSyncPath] --modsurl [ModsUrl]
This command will update automatically when you fill in both fields above.
